# Walter Bishop Labs – Calculus Experiments

A research-style repository exploring multivariable calculus
through visualization, coding experiments, and real-world models.

## Goals

• Build strong intuition for calculus concepts  
• Visualize mathematical ideas using Python  
• Connect calculus to real-world systems  
• Develop a laboratory approach to mathematics

## Concepts Covered

Continuity  
Derivatives  
Partial derivatives  
Gradients  
Optimization  
Differential equations

## How to Use This Repo

1. Start with notebooks for intuition
2. Explore visualizations
3. Run experiments
4. Modify models

## Example Visualizations

- Limit approaching a point
- Tangent line movement
- Gradient fields
- Surface plots
